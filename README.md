### How to start

- Clone project : 
```
git clone https://github.com/ronlechiht/typescript-training.git
```
- Move in practice folder: 
```
cd typescript-training/practice
```
- Checkout branch : 
```
git checkout dev
```
- Install packages: 
```
npm install
```
- Start project: 
```
npm start
```
- Open on web: http://localhost:1234
